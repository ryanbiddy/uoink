What is Uoink?

Uoink turns YouTube videos into structured input for Claude or ChatGPT — full transcript, screenshots at any interval, comments, and channel context, packaged into one paste-ready document.

Skip the "summarize this for me" routine. Hand the model the whole video and ask it anything.

2

Install the Uoink helper

Uoink runs a tiny program on your computer that does the heavy lifting locally — that's why your data stays private. Nothing leaves your machine except the YouTube transcript request.

Uoink starts automatically when you log in, runs silently in your system tray, and saves corpora to your Desktop's Uoink folder.

MCP

Agent Integration

MCP lets Claude Desktop, Cursor, and other AI agents call Uoink as a local tool: uoink a video, search saved corpora, inspect a transcript, or run comment and hook analysis.

Uoink does not edit your agent configs automatically. Copy the snippet for your client and paste it into that client's MCP settings. Stdio is the supported launch path; local HTTP JSON-RPC is experimental.

Skill

Uoink Operator Skill (recommended for Claude users)

The Skill turns your AI client into a YouTube creator research analyst when it works with Uoink corpora. It adds citation discipline, operator-grade analysis, tweet-mode hook autopsies, and the Uoink Hook Type taxonomy.

Use the one-liner for clients that support skills natively. For everything else, copy the system prompt below.
